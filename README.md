# bite

Ce site a été généré avec [Cocodex](https://cocodex.app). Il t'appartient entièrement.

## Démarrer en local

```bash
npm install
npm run dev
```

Puis ouvre [http://localhost:3000](http://localhost:3000).

## Construire pour la production

```bash
npm run build
```

Les fichiers statiques sont générés dans `out/`. Tu peux les déployer sur n'importe quel hébergeur statique (Netlify, Vercel, Cloudflare Pages, un simple serveur Nginx…).

## Personnaliser le contenu

Tout le contenu du site (textes, couleurs, sections) est dans `src/content.ts`. Modifie, sauvegarde, le site se met à jour.
